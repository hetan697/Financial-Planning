# 个人财务管理系统

一个基于Vue.js的个人财务管理应用程序，帮助您跟踪收支、管理投资并获得个性化的投资建议。

> 本项目基于vibe coding理念实现，强调代码的流畅性、一致性和良好的开发体验。

## 功能特性

### 1. 财务记录管理
- 添加收入和支出记录
- 查看交易历史
- 实时计算总收入、总支出和账户余额
- 支持自定义交易类型（收入/支出明细）

### 2. 投资组合管理
- 添加和管理投资项目（股票、基金、债券等）
- 跟踪投资数量、买入价格和当前价格
- 实时计算投资总价值和盈亏情况
- 支持自定义投资类型

### 3. 个性化投资建议
- 根据您的财务状况提供投资分配建议
- 包括紧急备用金、稳健型理财、基金投资、股票投资等配置建议
- 可视化资金分配比例
- 投资计算器支持自定义投资方式、收益率和收益计算

### 4. 数据持久化与管理
- 使用浏览器localStorage自动保存数据
- 数据在页面刷新或关闭后不会丢失
- 提供数据导入导出功能（JSON格式）
- 提供清除所有数据功能
- 类型数据（投资类型、交易类型）同样支持导入导出

### 5. 类型管理
- 支持自定义投资类型和交易类型
- 所有类型存储在localStorage中
- 提供图形化界面管理所有类型

## 技术栈

- Vue.js 3 (原生JavaScript，无需构建工具)
- HTML5
- CSS3 (含Flexbox和Grid布局)
- localStorage API (数据持久化)
- Element Plus (UI组件库)

## 快速开始

### 运行项目

1. 克隆或下载项目到本地
2. 在项目根目录下启动HTTP服务器:
   ```bash
   # 使用Python内置服务器
   python3 -m http.server 8000
   
   # 或使用npm脚本
   npm start
   ```
3. 在浏览器中打开 `http://localhost:8000`

### 使用说明

1. **数据看板**
   - 切换到"数据看板"标签页
   - 查看财务概览、收支趋势图、支出分类图和投资收益趋势图

2. **财务记录**
   - 切换到"财务记录"标签页
   - 点击"添加交易"按钮添加新的收入或支出记录
   - 可以编辑或删除已有记录

3. **投资管理**
   - 切换到"投资管理"标签页
   - 点击"添加投资"按钮添加新的投资项目
   - 可以编辑或删除已有投资项目

4. **投资建议**
   - 切换到"投资建议"标签页
   - 系统会根据您的当前财务状况提供个性化投资建议
   - 使用投资计算器自定义投资方式和收益率

5. **类型管理**
   - 切换到"类型管理"标签页
   - 可以自定义投资类型和交易类型（收入/支出明细）
   - 修改后点击"保存类型设置"生效

6. **数据管理**
   - 在"数据看板"中可以导出数据到JSON文件
   - 可以从JSON文件导入数据
   - 可以清除所有数据

## 投资建议算法

系统根据以下规则提供投资建议：

1. **资金充足时**（余额大于紧急备用金）:
   - 紧急备用金：基于过去6个月平均月支出×3个月
   - 稳健型理财：根据风险评估结果分配比例
   - 基金投资：根据风险评估结果分配比例
   - 股票投资：根据风险评估结果分配比例

2. **资金不足时**:
   - 建议优先建立紧急备用金

## 数据持久化

所有数据都保存在浏览器的localStorage中，具有以下特点：
- 数据在页面刷新或关闭后不会丢失
- 不同浏览器或设备间数据不共享
- 提供数据导入导出功能
- 提供"清除所有数据"按钮用于手动清空数据

## 开发说明

该项目使用Vue 3 Composition API构建，采用单文件组件(SFC)格式，但无需构建步骤即可直接在浏览器中运行。

项目基于vibe coding理念实现，注重：
- 代码的流畅性和一致性
- 良好的开发体验
- 组件间的和谐协作
- 用户界面的顺畅交互

## 关于vibe coding

vibe coding是一种编程理念，强调代码不仅要功能正确，还要传达正确的"氛围"或"感觉"。在本项目中体现为：
- 组件之间的一致性和协调性
- 用户界面的流畅交互体验
- 代码结构的清晰和优雅
- 功能模块间的自然衔接
