# 个人财务管理系统

一个基于React + Vite + Ant Design构建的现代化个人财务管理Web应用。提供收支管理、投资跟踪、数据可视化和财务分析功能。

## ✨ 主要特性

- **📊 财务看板**: 直观展示总资产、收支趋势、支出占比和投资收益概览。
- **💰 收支管理**: 轻松记录日常收入和支出，支持自定义分类和备注，提供列表管理和统计。
- **📈 投资管理**: 追踪股票、基金等投资项目的成本、当前价值和实时盈亏（支持+/-红绿涨跌色）。
- **🧮 投资计算器**: 自定义多种投资方式组合，实时计算加权平均收益率和资产分配计划。
- **🏷️ 类型管理**: 灵活配置收入、支出和投资的分类标签。
- **💾 数据安全**: 
  - 所有数据存储在本地浏览器（LocalStorage），隐私安全。
  - 支持JSON格式一键导出备份和恢复（导入）。
  - 支持数据一键清除。

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI 组件库**: Ant Design 5
- **图表库**: Chart.js + react-chartjs-2
- **数据处理**: Day.js, UUID
- **图标**: Ant Design Icons

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:5173` 查看应用。

### 构建生产版本

```bash
npm run build
```

## 📦 部署

本项目配置了 GitHub Actions，推送到 `main` 分支后会自动构建并部署到 GitHub Pages。

需要确保在 GitHub 仓库设置中开启 Pages 功能，并选择 `gh-pages` 分支作为源。

## 📝 使用指南

1. **首次使用**: 建议先在"类型管理"中配置适合自己的收支分类。
2. **记录数据**: 在"收支记录"页面添加日常流水。
3. **管理投资**: 在"投资管理"页面添加持仓信息，系统会自动计算盈亏。
4. **规划资产**: 使用"投资计算器"模拟不同的资产配置方案。
5. **数据备份**: 定期在"数据管理"页面导出JSON备份文件。
