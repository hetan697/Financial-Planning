# 个人财务管理系统

一个基于Vue.js的个人财务管理应用程序，帮助您跟踪收支、管理投资并获得个性化的投资建议。

## 功能特性

### 1. 财务记录管理
- 添加收入和支出记录
- 查看交易历史
- 实时计算总收入、总支出和账户余额

### 2. 投资组合管理
- 添加和管理投资项目（股票、基金、债券等）
- 跟踪投资数量、买入价格和当前价格
- 实时计算投资总价值和盈亏情况

### 3. 个性化投资建议
- 根据您的财务状况提供投资分配建议
- 包括紧急备用金、稳健型理财、基金投资、股票投资和保险配置建议
- 可视化资金分配比例

### 4. 数据持久化
- 使用浏览器localStorage自动保存数据
- 数据在页面刷新或关闭后不会丢失
- 提供清除所有数据功能

## 技术栈

- Vue.js 3 (原生JavaScript，无需构建工具)
- HTML5
- CSS3 (含Flexbox和Grid布局)
- localStorage API (数据持久化)

## 快速开始

### 运行项目

1. 克隆或下载项目到本地
2. 在项目根目录下启动HTTP服务器:
   ```bash
   # 使用Python内置服务器
   python3 -m http.server 8000
   
   # 或使用npm脚本
   npm start
   ```
3. 在浏览器中打开 `http://localhost:8000`

### 使用说明

1. **财务记录**
   - 切换到"财务记录"标签页
   - 在表单中填写交易类型、描述、金额和日期
   - 点击"添加"按钮添加记录
   - 可以通过每条记录后的"删除"按钮删除记录

2. **投资管理**
   - 切换到"投资管理"标签页
   - 在表单中填写投资类型、名称、数量、买入价格等信息
   - 点击"添加投资"按钮添加投资项目
   - 可以通过每条记录后的"删除"按钮删除投资项目

3. **投资建议**
   - 切换到"投资建议"标签页
   - 系统会根据您的当前财务状况提供个性化投资建议
   - 建议包括资金分配比例和具体金额

## 投资建议算法

系统根据以下规则提供投资建议：

1. **资金充足时**（余额大于紧急备用金）:
   - 紧急备用金：3个月生活费
   - 稳健型理财：30%
   - 基金投资：40%
   - 股票投资：20%
   - 保险配置：10%

2. **资金不足时**:
   - 建议优先建立紧急备用金

## 数据持久化

所有数据都保存在浏览器的localStorage中，具有以下特点：
- 数据在页面刷新或关闭后不会丢失
- 不同浏览器或设备间数据不共享
- 提供"清除所有数据"按钮用于手动清空数据
